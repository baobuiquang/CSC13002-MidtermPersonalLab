### Local

1. Clone this repository
2. Install Python
3. Install libraries/packages in `crm1/requirements.txt`
4. Terminal:
   1. `cd` to crm1
   2. `python manage.py runserver`
5. Open `localhost` port `8000` (http://127.0.0.1:8000)
